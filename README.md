# lyrics-timestamp-generator
A modern Python + Tkinter tool for generating accurate lyric timestamps synced with audio playback.
This project provides a graphical tool that lets you load an audio file, input lyrics, and record the exact timestamp for each lyric line in real time as the music plays. The final result is a clean, well-structured JSON file that can be used for lyric syncing, karaoke apps, audioâ€“text alignment tools, and more.
